# SDCM_parser
Трогать в таблице ничего не надо. Все значения и названия сами сохранятся, а графики построятся.

Порядок действий
---
1. Установить python и необходимые библиотеки.
2. Скачать `parser_wrapper.py` и `лаба.xlsx`.
3. Настроить и запустить `parser_wrapper.py`.
4. Открыть результат в `лаба.xlsx`.

Файлы должны быть в одной папке.

Установка
---
1. Скачиваете с сайта установщик и, собственно, устанавливаете.
   - Для удобства создайте и укажите для установки отдельную папку на диске `C`.
   - Обязательно поставьте галочку возле `add to PATH`.
2. Нажимаете `Win+R`.
3. Вводите `cmd`, затем `pip install requests pandas openpyxl`.

Настройка
---
Обязательно посетите сайт СДКМ, чтобы проверить, работают ли станции. Если некоторые станции из `STATIONS_INFO` в `parser_wrapper.py` отключены, то укажите свои:
1. Находите на карте станцию недалеко от нужной.
2. Ищете в таблице нужный код станции. Я хз, почему они не указали и названия, и код.
3. Жмёте ПКМ по `БПУ1` и копируете адрес ссылки. В самом её конце указаны ID станции и приёмника.
4. Записываете название станции и ID вместо неработающей.

Запуск
---
Всё в том же окне `cmd` пропишите `cd /d Ваш:\путь\к\файлу & python parser_wrapper.py`.
Путь к файлу копируйте в свойствах файла.
