# SDCM_parser
Предназначена для решения лабораторной работы. Я бы мог оставить сразу значения в таблице или вообще файл, но как же не заставить вас что-то делать самим?

-----
### Порядок действий
1. Установить python и необходимые библиотеки.
2. Скачать `parser_wrapper.py` и `лаба.xlsx`.
3. Запустить `parser_wrapper.py`.
4. Открыть результат в `лаба.xlsx`.
-----
### Установка
1. Скачиваете с сайта установщик.
2. Собственно, установите python.
   - Для удобства доступа укажите создайте отдельную папку.
   - Не забудьте поставить галочку возле `add to PATH`.
3. Нажимаете `Win+R`
4. Вводите `cmd`
5. Вводите `pip install requests pandas openpyxl`
