# SDCM_parser
Трогать в таблице ничего не надо. Все значения и названия сами сохранятся, а графики построятся.

Порядок действий
---
1. Установить python и необходимые библиотеки.
2. Скачать `parser_wrapper.py` и `лаба.xlsx`. Файлы должны храниться в одной папке.
3. Настроить и запустить `parser_wrapper.py`.
4. Открыть результат в `лаба.xlsx`.

Установка
---
1. Скачайте с сайта установщик python и, собственно, установите.
   - Для удобства создайте и укажите для установки отдельную папку на диске `C`.
   - Обязательно поставьте галочку возле `add to PATH`.
2. Нажмите `Win+R`.
3. Введите `cmd`, затем `pip install requests pandas openpyxl`.

Настройка
---
Обязательно посетите сайт СДКМ, чтобы проверить, работают ли станции. Если некоторые станции из `STATIONS_INFO` в `parser_wrapper.py` отключены, то укажите свои:
1. Найдите на карте станцию недалеко от нужной.
2. Найдите в таблице нужный код станции. Наведите на него курсор и ждите всплывающей подскаски с именем станции.
3. Нажмите ПКМ по `БПУ1` и скопируйте адрес ссылки. В самом её конце указаны ID станции и приёмника.
4. Запишите название станции и ID вместо неработающей.

Запуск
---
1. Скопируйте путь к `parser_wrapper.py` в свойствах файла.
2. В окне `cmd` введите `cd /d Ваш:\путь\к\файлу & python parser_wrapper.py`.
