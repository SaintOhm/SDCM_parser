# SDCM_parser
Предназначена для решения лабораторной работы.

Я бы мог оставить сразу значения в таблице или вообще `.docx`, но как же не заставить вас что-то делать самим? Я постратил своё время, теперь и вы своё потратьте. Полчаса не так много.

Трогать в таблице ничего не надо. Все значения сами сохранятся.

Порядок действий
---
1. Установить python и необходимые библиотеки.
2. Скачать `parser_wrapper.py` и `лаба.xlsx`.
3. Настроить и запустить `parser_wrapper.py`.
4. Открыть результат в `лаба.xlsx`.

Файлы должны быть в одной папке.

Установка
---
1. Скачиваете с сайта установщик.
2. Собственно, установливаете.
   - Для удобства создайте и укажите отдельную папку.
   - Не забудьте поставить галочку возле `add to PATH`.
3. Нажимаете `Win+R`
4. Вводите `cmd`
5. Вводите `pip install requests pandas openpyxl`

Настройка
---
Обязательно посетите сайт, чтобы проверить, работают ли станции. Если станции в `parser_wrapper.py` отключены, то укажите свои:
1. Находите на карте станцию недалеко от нужной.
2. Ищете в таблице нужный код станции. Я хз, почему нельзя было указать и названия и код в разных столбцах.
3. Наводите курсор на `БПУ1`
4. Жмёте ПКМ
5. Жмёте `Копировать адрес ссылки`
6. В самом её конце указаны ID станции и приёмника. Записываете их в `STATIONS_INFO` вместо не работающей станции. Про её название не забудьте.
